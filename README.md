# Portfólio — Lucas Stanguerlin

Site de apresentação de serviços de desenvolvimento web.

## Abrir localmente

```powershell
cd portfolio
npx --yes serve .
```

Acesse `http://localhost:3000` no navegador.

Ou abra `index.html` diretamente no Chrome/Edge.

## Publicar online (grátis)

- [Netlify Drop](https://app.netlify.com/drop) — arraste a pasta `portfolio`
- [Vercel](https://vercel.com) — importe o projeto
- Hospedagem compartilhada — envie os arquivos via FTP

## Estrutura

- `index.html` — site principal
- `demos/` — 4 mini-sites de demonstração por categoria
- `assets/logo.png` — sua logo
